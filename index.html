Website adalah kumpulan halaman web yang saling terhubung dan dapat diakses melalui internet. website seperti sebuah buku, di mana setiap halamannya berisi informasi yang berbeda-beda. Namun, buku ini tidak menggunakan kertas, melainkan dari Bahasa komputer.Setiap halaman dalam buku ini memiliki alamat unik yang disebut URL, sehingga kita bisa menemukannya dengan mudah. Website biasanya dibuat menggunakan bahasa pemrograman seperti HTML, CSS, dan JavaScript 3 Bahasa pemrograman ini adalah inti dari sebuah website.


HTML, CSS, dan JavaScript adalah inti sebuah website, tanpa 3 bahasa pemrograman ini kita tidak dapat menciptakan situs web yang tidak hanya informatif, tetapi juga menarik dan mudah digunakan.


  • Cara menintegrasi HTML dalam pemrograman web dilakukan dengan menuliskan tag-tag HTML yang akan diterjemahkan oleh browser menjadi elemen-elemen visual seperti judul, paragraf, gambar, dan tautan.

  • Cara mengintegrasikan CSS dalam sebuah website adalah dengan menghubungkan file CSS ke file HTML. Kita bisa melakukan ini dengan menggunakan tag link di dalam bagian head dari file HTML. Tag link ini memiliki atribut rel="stylesheet" yang menandakan bahwa file yang terhubung adalah sebuah stylesheet, dan atribut href yang menunjuk ke lokasi file CSS

  • Cara mengintegrasikan JavaScript ke dalam sebuah website, kita dapat menempatkan kode JavaScript langsung di dalam tag script pada file HTML Anda. javascript akan dieksekusi saat halaman web dimuat. Selain itu, kita juga dapat membuat file JavaScript terpisah dengan ekstensi .js, debgan menggunakan tag script dengan atribut src yang menunjuk ke lokasi file JavaScript tersebut.
